# My X-Window Manager

- [Alt]+[Enter]     - Launch terminal
- [Alt]+[b]         - Launch browser
- [Alt]+[number]    - Go to workspace `number` 
- [Alt]+[Tab]       - Go to previous workspace
- [Alt]+[n]         - Go to next window
- [Alt]+[m]         - Move mouse to next screen
- [Alt]+[Shift]+[c] - Kill current window
- [Alt]+[Shift]+[q] - Exit
